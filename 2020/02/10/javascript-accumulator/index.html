先來說說 curry 是什麽樣的東西
『透過部分參數呼叫一個 function，然後讓此 function 回傳 function 去處理剩餘的參數』
以下先來個符合題目的範例

這種 code 根本不是人看的，接下來就需要另一個概念『遞迴』

實作 - 遞迴

遞迴的概念就是重複呼叫 function 本身，然後達到某個條件在停止
所以關鍵在於『需要讓他一直呼叫 function 直到呼叫 sumOf 才停止』
